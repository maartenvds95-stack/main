Sneltoetsen & Help

  • Spatiebalk (vasthouden) / Muiswiel-klik: Pannen (schuiven)
  • Ctrl + Scroll: In-/Uitzoomen
  • Shift (tijdens tekenen): Orthogonaal (90 graden) lijnen
  • Enter of Escape: Oppervlakte meting afsluiten
  • Delete / Backspace: Selectie verwijderen
  • Ctrl + Z: Ongedaan maken (Undo)
  • Ctrl + C / Ctrl + V: Kopiëren en plakken
  • Ctrl + G / Ctrl + Shift + G: Groeperen / Degroeperen
  • R: Object 90 graden roteren
